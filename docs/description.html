Bootstrap 3 based template for Subrion CMS Autos Premium Package.

Template features:

  1. Mobile friendly
  2. Ability to change background image on home page in admin panel
  3. Two color schemes: light and dark
  4. Social links configuration

Configurations:

  • Change top address and phone: got to Language management and find phrases by keys: website_address, website_phone
  • Social links configuration: go to Template management

More updates coming soon.
